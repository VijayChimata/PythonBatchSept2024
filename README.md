## Git Commands

### To clone a repo,
    git clone https URL

### To stage the changes
    git add <filename>

### To commit the changes
    git commit -m "Commit Message"

### To check the status 
    git status

### To push the changes
    git push origin <SourceBranch>

    class01 -> main


### To check/modified content in the existing file,
    git diff <filename>
